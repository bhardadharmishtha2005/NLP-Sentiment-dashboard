# NLP-Sentiment-dashboard

## Frontend and Backend Technologies:
The dashboard utilizes Streamlit for a responsive web interface, incorporating HTML and CSS for custom UI elements. The backend is powered by Python, leveraging the TextBlob library for Natural Language Processing (NLP) and Pandas for high-speed data manipulation and analysis.

## Platform and Tools: 
The application is developed on Microsoft Windows using VS Code as the Integrated Development Environment (IDE). Regular Expressions (Regex) are implemented within the backend to ensure clean data ingestion and high processing accuracy.

## Key Features: 
Includes a Batch Data Processor with automated encoding detection (UTF-16/Latin-1) to handle diverse file formats, and an interactive Keyword Search Filter for granular analysis of specific product trends within large-scale datasets.
